## Реализованные функции
1. **Переключатель языков RU/AM** — полная локализация всех текстов
2. **Автоматический калькулятор услуг** — 18 услуг разбиты по 4 категориям:
   - Выкупы (2 услуги)
   - Отзывы (3 услуги)
   - Фотосъёмка (6 услуг)
   - Логистика (7 услуг)
3. **Форма заявки** — сбор имени, контакта, товара, услуги, комментария → отправка в Telegram
4. **Прайс-лист** — все 18 услуг с ценами в ֏ AMD
5. **Галерея склада** с Lightbox
6. **Секция гарантий** с личной гарантией Георгия Дарбиняна
7. **Сравнительная таблица** Go to Top vs конкуренты
8. **FAQ аккордеон** — 7 вопросов
9. **5 шагов** — визуальная воронка процесса
10. **Тикер-строка** — бесконечная прокрутка преимуществ
11. **Плавающая кнопка Telegram**
12. **Exit-intent popup** — перехват уходящих посетителей
13. **Анимации** — fade-up при скролле, счётчики
14. **Адаптивность** — мобильное меню, tablet и desktop

## Технический стек
- **Backend**: Hono (TypeScript) на Cloudflare Workers
- **Frontend**: HTML + CSS + JS (inline, single-page)
- **CDN**: Tailwind-like custom CSS, Font Awesome, Inter font
- **Deploy**: Cloudflare Pages
- **Dev**: Vite + Wrangler

## API эндпоинты
- `GET /` — основная страница
- `GET /api/health` — проверка статуса
- `POST /api/lead` — приём заявок (JSON: name, contact, product, service, message)

## Условия работы (отображаются на сайте)
- Крупный товар: >3 кг или сторона >55 см. >10 кг — индивидуально
- Отзывы: не более 50% от выкупов
- Защитные пломбы после съёмки не восстанавливаются
