# Go to Top — Админ-платформа PRD

## Исходный проект
- **Go to Top** — сервис продвижения товаров на Wildberries через выкупы
- Оригинальный стек: Hono (TypeScript) + Cloudflare D1 + Pages
- GitHub: https://github.com/Gor1x1/Go-to-Top-Lending-genspark
- Сайт: https://gototop.win
- Оригинальная админка: https://gototop-wb.pages.dev/admin

## Новый стек (Emergent Platform)
- **Backend**: FastAPI (Python) + MongoDB
- **Frontend**: React 18 + React Router 6 + Lucide Icons
- **Auth**: JWT (python-jose) + bcrypt
- **Дизайн**: Dark UI, фирменный фиолетовый (#8B5CF6), шрифт Manrope

## Персоны
- **Главный Админ** — полный доступ, управление сотрудниками и доступами
- **Разработчик** — дашборд, блоки, калькулятор, скрипты, настройки
- **Аналитик** — дашборд, лиды
- **Оператор** — дашборд, лиды
- **Выкупщик** — дашборд
- **Курьер** — дашборд

## 14 разделов админки
1. Дашборд — статистика, последние заявки, журнал действий
2. Лиды / CRM — фильтрация, статусы, назначение ответственных, CSV экспорт
3. Конструктор блоков — порядок, тексты (RU/AM), видимость секций сайта
4. Калькулятор — вкладки и услуги с ценообразованием
5. PDF шаблон — коммерческое предложение
6. Реферальные коды — промокоды, скидки, бесплатные отзывы
7. Счётчики слотов — urgency-элементы для сайта
8. Футер сайта — контакты, соцсети, копирайт (RU/AM)
9. TG сообщения — шаблоны для кнопок на сайте
10. TG Бот / Уведомления — конфигурация бота, тест-отправка
11. Скрипты — аналитика, пиксели, meta-теги
12. Сотрудники — CRUD, роли, деактивация, сброс пароля
13. Доступы — настройка разделов для каждого сотрудника
14. Настройки — смена пароля, профиль

## Реализовано (2026-02-19)
### Фаза 1: Система ролей и управление
- JWT авторизация, 6 ролей, CRUD сотрудников, настройка доступов, журнал активности

### Фаза 1.5: Объединение с оригинальной админкой
- Все 14 разделов из оригинала перенесены на React + FastAPI + MongoDB
- Двуязычная поддержка RU/AM во всех разделах контента
- CSV экспорт лидов, тест Telegram бота
- Публичный endpoint /api/lead для приёма заявок с лендинга
