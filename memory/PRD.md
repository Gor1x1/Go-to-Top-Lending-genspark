# Go to Top — Админ-платформа PRD

## Исходный проект
- **Go to Top** — сервис продвижения товаров на Wildberries через выкупы
- Оригинальный стек: Hono (TypeScript) + Cloudflare D1 + Pages
- GitHub: https://github.com/Gor1x1/Go-to-Top-Lending-genspark
- Живой сайт: https://gototop-wb.pages.dev

## Новый стек (Emergent Platform)
- **Backend**: FastAPI (Python) + MongoDB
- **Frontend**: React 18 + React Router 6 + Lucide Icons
- **Auth**: JWT (python-jose) + bcrypt
- **Дизайн**: Dark UI, фирменный фиолетовый (#8B5CF6)

## Пользовательские персоны
- **Главный Админ** — полный доступ ко всему, управление сотрудниками и доступами
- **Разработчик** — дашборд, контент, калькулятор, аналитика, настройки
- **Аналитик** — дашборд, лиды, аналитика
- **Оператор** — дашборд, лиды, заказы
- **Выкупщик** — дашборд, заказы
- **Курьер** — дашборд, заказы

## Ключевые требования (статические)
1. Многопользовательская система с 6 ролями
2. Главный админ управляет доступами и сотрудниками
3. CRM для обработки лидов с лендинга
4. Журнал действий всех пользователей
5. MVP — бесплатные сервисы, без платных интеграций

## Что реализовано (Фаза 1 — 2026-02-19)
- **Система авторизации**: JWT login, change password, токен-верификация
- **6 ролей**: main_admin, developer, analyst, operator, buyer, courier  
- **Управление сотрудниками**: создание, редактирование, удаление, сброс пароля, деактивация
- **Управление доступами**: настройка разделов для каждого сотрудника
- **CRM Лиды**: фильтрация по статусам, назначение ответственных, заметки, удаление
- **Дашборд**: статистика лидов, счётчик сотрудников, последние заявки, журнал действий
- **Публичный endpoint**: /api/lead для приёма заявок с лендинга
- **Журнал активности**: все действия логируются
- Дефолтный админ: admin / gototop2026
